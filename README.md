## Installation

Install with [yarn](https://github.com/yarnpkg/yarn)

```bash
yarn add serverless-plugin-api-gateway-integrations --dev
```

or [npm](https://www.npmjs.com/)

```bash
npm install serverless-plugin-api-gateway-integrations --save-dev
```
